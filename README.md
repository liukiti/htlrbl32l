# iMCP HTLRBL32L - SDK

**PRELIMINARY SDK** for iMCP HTLRBLR32L.

## How to Clone this Branch

Command to clone this branch:
```
git clone --single-branch --branch SDK git@github.com:htmicron/htlrbl32l.git
```
